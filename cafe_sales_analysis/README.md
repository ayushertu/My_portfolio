# Cafe Sales Analysis

## Project Overview
This project is designed to analyze sales data from a café, providing insights into sales trends, customer preferences, and overall performance. The analysis is aimed at helping café management make informed business decisions based on data.

## Project Structure
- **cafe_sales_analysis**: Contains all the analysis files and notebooks
- **data**: Directory for storing datasets used in the analysis
- **notebooks**: Jupyter notebooks with detailed analysis steps and visualizations
- **dashboard**: Dashboard files showcasing key metrics and insights

## Data
The datasets used for this analysis include:
- Sales data: Contains transaction records, including timestamps, items sold, and revenue generated.
- Customer data: Information on customers, including demographics and purchase history.

## Analysis Notebooks
The analysis notebooks include:
1. **Sales_Analysis.ipynb**: Detailed analysis of sales trends over time.
2. **Customer_Analysis.ipynb**: Examination of customer preferences and behaviors.
3. **Marketing_Analysis.ipynb**: Insights into the effectiveness of marketing campaigns.

## Dashboard
The dashboard presents a visual summary of the café's performance over specified periods. It includes charts and tables displaying key performance indicators (KPIs) such as total sales, average transaction value, and customer acquisition trends.

## Conclusion
This project aims to provide actionable insights that can help improve the cafe's operational strategies and ultimately increase profitability.
